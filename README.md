# Automatizador de Planilhas — Validação e Correção de Dados

Aplicação local desenvolvida como **projeto de portfólio** para validação, detecção e correção de inconsistências em planilhas **CSV** e **XLSX**.

O projeto simula uma solução aplicável ao mercado corporativo, com foco em **qualidade de dados**, **automação** e **redução de retrabalho manual**.

---

##  Problema Abordado
Planilhas utilizadas em contextos reais frequentemente apresentam:
- CPFs e CNPJs inválidos ou mal formatados
- Datas em múltiplos padrões
- Campos numéricos inconsistentes
- Textos com variações e ruídos

Esses problemas geram falhas em sistemas, retrabalho e perda de tempo operacional.

---

##  Objetivo do Projeto
Demonstrar, na prática:
- Validação e saneamento de dados
- Automação de correções recorrentes
- Organização de um backend orientado a problemas reais
- Estrutura pensada para evolução futura (API / SaaS)

---

##  Funcionalidades
-  Detecção de inconsistências em **CPF** e **CNPJ**
-  Validação e padronização de **datas**
-  Correção de campos **textuais**
-  Verificação de **valores numéricos**
-  Geração de relatório com problemas identificados
-  Exportação de arquivos corrigidos
-  Execução totalmente local

---

##  Arquitetura e Decisões Técnicas
- Backend desacoplado do frontend
- Processamento de dados com **Pandas**
- Validações organizadas de forma modular
- Execução local para preservar dados sensíveis
- Estrutura preparada para futura expansão

---

##  Tecnologias Utilizadas
- **Python 3**
- **Flask**
- **Pandas**
- **HTML / CSS / JavaScript**
- Suporte a **CSV** e **XLSX**
